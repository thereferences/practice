# Machines

:::{note}
Each colleague has an HP ZBook Fury 16 G10 Mobile Workstation; selected from the organisation's HP brochure, and purchased via digital purchasing.  Specification Excerpt: NVIDIA RTX 2000 Ada 8GB Graphics, Intel Core i7-13700HX Processor, 32GB (2 x 16GB) DDR5 4800 <span class="tooltip">SO-DIMM<span class="tooltiptext">Small Outline Dual In-line Memory Module</span></span>
Memory, 2TB <abbr title="Peripheral Component Interconnect Express">PCIe</abbr>-4x4 2280 <abbr title="Non-Volatile Memory Express">NVMe</abbr> <abbr title="self-encrypting drive">SED</abbr> OPAL2 TLC SSD
:::

<br>

All the organisation's computer usage rules apply; remember, Universal Serial Bus (USB) sticks are disallowed.

<br>

**Initial Set-up**

Foremost:

* Do not select Microsoft Account.  Instead, select the **School/Business** option, thereafter select **other domains** option.
* Install and set up the security software.
* Hard Drive Partitioning: Consider partitioning the C drive into two drives; such that the second drive is for development purposes.

<br>

Additionally, install

* [firefox](https://www.mozilla.org/en-GB/firefox/)
* [opera](https://www.opera.com)
* [KeePassXC](https://keepassxc.org/)

within Windows.  Ensure that the privacy & security settings of the browsers are strict; additionally, set the default search engine to _duckduckgo_.
